Memoria

En informática, la memoria es el dispositivo que retiene, memoriza o almacena datos informaticos durante algún periodo de tiempo

La memoria proporciona una de las principales funciones de la computación moderna: el almacenamiento de información y conocimiento. Para ser ejecutado los programas y datos, estos deben ser almacenados en la memoria del computador. El procesador buscará cada instrucción a ejecutar y los datos con los cuales ira a operar en la memoria del sistema. El general, de forma que esta sea accesible para la CPU. Eñ sistema de memoria de los ordenadores modernos consta de varias secciones con diferentes tareas:

  • La memoria de trabajo o RAM (Random Acces Memory - Memoria de Acceso Aleatorio).

  • La memoria cache.
  • La memoria CMOS.
  • La ROM (Read Only Memory - Memoria de solo lectura)
  • La memoria grafica o de video
